When looking at the nmap, I get the idea that the box os is Windows IOT. I got to the http service on port 8080 but it needs credentials.

I’ve never done a ctf on windows iot so I google a few things and eventually google windows iot rce and find a called SirepRAT), using this, I put the ip of the box in and it works!:

I also upgrade to a better shell by uploading nc64.exe from the box with powershell from my python http server and call the binary using the SirepRAT.py

There are no users in C:\Users aside from Public but the rest of the users can be found at C:\Data\Users\

I can go into administrator and read root.txt however it seems to be encrypted:

This is the same for user.txt so we need a way to change users/escalation privileges. I tried some decryption commands I found via google but nothing worked since it has to be ran from the owner’s account. I cannot read anything useful in the users directories so I look at programs that are running inside C:\Program Files and find there’s only one folder, WindowsPowerShell

There is a file called r.bat inside modules\packagemanagement but it is hidden so can be found with ls -Force


@echo off



:LOOP



for /F "skip=6" %%i in ('net localgroup "administrators"') do net localgroup "administrators" %%i /delete



net user app mesh5143

net user administrator _1nt3rn37ofTh1nGz



ping -n 3 127.0.0.1



cls



GOTO :LOOP



:EXIT

There are two usernames and passwords:


app:mesh5143

administrator:_1nt3rn37ofTh1nGz

Going back to port 8080, I try the app credentials and get access!:

I then can go to the processes tab where I can run a command:

When I re-run the nc binary I previously uploaded, I get another shell but this is with a the app user so can now read the directory:

I found that now I could decrypt user.txt by importing the credential inside user.txt and then using it with another command to read the contents:

This imports the file into $credential and then:


$Credential = Import-CliXml -Path "C:\Data\Users\app\user.txt"



$credential.GetNetworkCredential().Password

Repeat this for admin and get root.txt
